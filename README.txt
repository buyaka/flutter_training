Flutter апп хөгжүүлэлт хийнгээ суралцах бүлгэмийн хичээлд зориулан үүсгэв

Прожект бүтэц

backend  :　Rails api(6.0) 
frontend :  Flutter
Procfile : Process management tool file
setup.sh : Prepare basic environment
start.sh : Run backend & frontend

Прожектоо клон хийж аваад 
1. setup.sh ажиллуулах, нээгээд доторх агуулгыг уншиж суулгах шаардлагатай зүйлсийн өөрсдөө суулгах 
2. start.sh
